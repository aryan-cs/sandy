0-to-1 photoreal walkthrough

Real capture first. Map extrusion second.

This launcher opens the official SuperSplat Viewer against a real captured Gaussian splat scene. It starts in Fly mode with WASD / arrow-key movement; the local scene also includes generated voxel and GLB collision assets.

Open Walkthrough Open Walk Mode Public Asset

Why this replaces the old MVP: the old OSM scene only had inferred building massing and top-down imagery. This uses photoreal radiance-field data, which is the visual layer needed for a digital twin-like walkthrough.

Current limitation: the generated collision works as a first pass, but capture-specific tuning is still required before Walk mode feels production-grade. Fly mode is the reliable MVP control path today.
